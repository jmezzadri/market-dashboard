## What this project is

A personal market risk dashboard built in React (Vite). It tracks 25 macro/financial indicators, computes a weighted composite stress score, and maps it to a 4-level conviction regime (LOW / NORMAL / ELEVATED / EXTREME) with corresponding portfolio positioning guidance.

The dashboard is live on Vercel. Every time `src/App.jsx` is edited and pushed to GitHub, Vercel auto-redeploys in ~60 seconds.

Daily data is auto-fetched at 7:00 AM every morning via a macOS LaunchAgent running `fetch_indicators.py`, which pulls from FRED API + Yahoo Finance and patches `src/App.jsx` in place.

## App.jsx structure — key data blocks

All data lives at the top of `src/App.jsx` in this order:

### 1. `SD{}` — Standard deviation baselines
```
id: { mean, sd, dir }
dir: "hw" = higher is worse, "lw" = lower is worse, "nw" = neutral
```
Used to convert raw values to a stress score (z-score relative to historical mean).

### 2. `AS_OF{}` — Last-updated dates per indicator
Date strings displayed in the UI. Auto-updated by the daily fetch for live indicators. Update manually for CAPE, ISM, and Bank Unrealized Losses.

### 3. `IND{}` — The 25 indicators (main data)
Each entry:
```
id: [displayName, categoryName, catKey, tier, unit, decimals,
     NOW, 1M_ago, 3M_ago, target1, target2, isLowGood,
     "description text", "current analysis text"]
```
- **Position [6] = NOW value** — the current live reading
- **Position [7] = 1M ago** — value 30 days prior
- **Position [8] = 3M ago** — value 90 days prior
- Positions [9] and [10] = benchmark reference levels

### 4. `WEIGHTS{}` — Indicator weights in composite score
Higher weight = more influence on the composite stress number.

### 5. `CONVICTION[]` — 4 regime levels
Each level has: score range, color, suggested allocation (eq/bd/ca/au %), and action text.
Levels: LOW (green) → NORMAL (yellow) → ELEVATED (amber) → EXTREME (red)

### 6. `ACCOUNTS[]` — Joe's portfolio
4 accounts tracked: Taxable Brokerage (…4471), 401k (…8823), Roth IRA (…2290), HSA (…7734).
Each position: `{ticker, name, value, price, shares, sector, beta, color, analysis}`

### 7. `ANALYSIS` string — Narrative commentary
Multi-line text block with current regime summary, what changed, and key risks for the next 4–8 weeks. Updated manually.
